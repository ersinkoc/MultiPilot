# Icons

Place your application icons here:

- `32x32.png` - 32x32 icon
- `128x128.png` - 128x128 icon
- `128x128@2x.png` - 256x256 icon (for retina)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

You can generate these from a single PNG/SVG using Tauri's icon generator:
```bash
npm run tauri icon /path/to/icon.png
```
